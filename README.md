## 基于知识图谱的课程推荐系统

**项目简介**

本项目基于知识图谱技术，开发了一个基于知识图谱的课程推荐系统。系统采用django框架，底层图谱数据库采用neo4j软件存储，并自带了一份课程数据数据，可以直接初始化进入neo4j。

**主要功能**

- 用户登录、注册、退出系统等功能。
- 查看课程列表，可以搜索课程的名字。
- 查看课程详情页，可以查看视频。
- 基于浏览历史，推荐相关课程。 
- 管理员可以对用户、课程等进行增删改查。


**技术架构**

- 前端：Django
- 后端：Neo4j
- 课程数据：CSV


**使用说明**

1. 下载项目代码。
2. 运行项目。
3. 登录系统，进行基本操作。
